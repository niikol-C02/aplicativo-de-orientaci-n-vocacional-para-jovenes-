# aplicativo-de-orientaci-n-vocacional-para-jovenes-
Nuestra app simplifica la orientación vocacional mediante herramientas de autoconocimiento y una base de datos actualizada de ofertas académicas. Diseñada para estudiantes que buscan claridad, ofrece perfiles detallados de profesiones, proyecciones de mercado laboral y mentoría personalizada para convertir el talento en una trayectoria exitosa.
